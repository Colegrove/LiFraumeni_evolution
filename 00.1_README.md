# Script Organization README
This template is divided into several sections; some of these are not meant to 
be modified by the end user.  The sections are:

00: Master control and information
01: Setup code
01.1: Functions
01.2: Color Scales
01.3: Font / figure size parameters
02: File loading
02.1: Depth
02.2: COSMIC
02.3: NoSelection
02.4: Samples MAF
03: Primary analysis
03.1: Mutation Frequencies
03.2: Mutation Composition Analyses
04: Secondary analysis
10+: Plots, tables, and other analyses

In most cases, the user should only modify sections 01.2, 01.3, 02.4.2, and 
04+, as well as adding new lines to the end of 00.0 as necessary.  All other 
sections should remain pretty much fixed.  Many modifications can be 
accomplished using a combination of input files and the config file 
(../processing_config.txt).

In case you feel the need to modify other parts, please keep in mind that 
modifications to the basic loading functions should be general enough to apply 
to all the different source files (Samples MAF, Cosmic, and No Selection).  
Detailed descriptions of each of the various steps appears below.  

## Detailed Descriptions

### 00.0: The Main Script

This script serves the same function as the "main" function in most programs: 
it is the script that runs all the others (either directly or indirectly).  
Modifications should be restricted to adding new modules as needed, and 
adding new scripts created by the user to allow for them to be run in 
sequence with the rest of the scripts.  

### 01.1: Main functions script

This script will load all the other functions.  In this template, I have taken 
the philosophy that, in general, every function should have its own script 
associated with it.  Each script should also include a brief description of 
what that script does, and should be well commented.  

### 01.2: Color Scales

The main purpose of this script is to allow color scales to be used in a 
uniform manner for the same variable across multiple figures within a study.  

### 01.3:  Font / figure size parameters

This should match the target journal, and give the figure size specification 
(in mm), as well as the font size specifications given by the journal.
